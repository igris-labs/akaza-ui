## Install the package

::code-group

```bash [npm]
npm install akaza-ui
```

```bash [pnpm]
pnpm add akaza-ui
```

```bash [yarn]
yarn add akaza-ui
```

::

## Import component styles

Akaza UI ships minimal CSS for animations and behavior (no visual styles). Import it once in your app entry point.

```ts [main.ts]
import 'akaza-ui/dist/akaza-ui.css'
```

For **Nuxt**, add it to your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  css: ['akaza-ui/dist/akaza-ui.css'],
})
```

## Import a component

All components are named exports from `akaza-ui`:

```vue
<script setup lang="ts">
import { Dialog, Tooltip, Tabs } from 'akaza-ui'
</script>
```

Auto-imports are not set up by default — import explicitly so your editor and TypeScript can resolve types correctly.

## TypeScript support

Akaza UI is fully typed. Types are exported alongside components:

```ts
import type { AccordionItem, DialogProps } from 'akaza-ui'
```
